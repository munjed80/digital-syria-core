# MVP SCOPE

## Included

- Unified backend foundation for citizen service requests pilot.
- Roles: citizen, employee, supervisor, admin.
- Core workflow: registration -> request submission -> review/status updates -> monitoring/admin management.
- Service catalog, requests, status history, internal notes, audit logs, notifications persistence.
- OpenAPI documentation via FastAPI.

## Not yet included

- Full production frontend implementation for all portals.
- External identity provider integration (future Keycloak/SSO replacement path prepared).
- Advanced analytics and BI reporting.
- Multi-governorate scaling and inter-agency federation.
